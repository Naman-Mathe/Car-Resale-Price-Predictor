# 🚗 Car Resale Price Predictor

An AI-powered web dashboard that predicts the resale value of used cars in the Indian market. Built with **Streamlit**, **scikit-learn**, and **Plotly** for a fully interactive experience.

---

## 📸 Features

- 🔮 **Price Prediction** — Enter car details and get an instant AI-estimated resale price
- 📊 **Interactive Charts** — Plotly-powered charts: price distribution, fuel split, brand rankings, age vs price scatter
- 🎯 **Gauge & Comparison** — See your predicted price on a gauge and compare it against market percentiles (P25 / P50 / P75)
- 🚘 **1,491 Car Models** — Searchable dropdown built from the real dataset; brand is auto-detected
- 📋 **Similar Listings** — Benchmark your prediction against real dataset entries for the same brand and fuel type
- 🌙 **Dark Automotive Theme** — Custom-styled UI with amber accents, no generic defaults

---

## 🗂️ Project Structure

```
project/
│
├── data/
│   └── car_data.csv          # Raw dataset (4,340 listings)
│
├── app.py                    # Streamlit dashboard (main app)
├── train_model.py            # Model training + EDA script
│
├── model.pkl                 # Trained LinearRegression model (generated)
```

> `model.pkl` and `columns.pkl` are generated by running `train_model.py` and are **not** included in the repo.

## ⚙️ Setup & Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/car-price-predictor.git
cd car-price-predictor
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Train the model

```bash
python train_model.py
```

This will generate `model.pkl` and `columns.pkl` in the project root.

### 4. Run the app

```bash
streamlit run app.py
```

Open your browser at `http://localhost:8501`

## 📊 Dataset

| Property | Details |
|---|---|
| File | `data/car_data.csv` |
| Rows | 4,340 listings |
| Source | Indian used car market |
| Target column | `selling_price` |

**Columns in the dataset:**

| Column | Description |
|---|---|
| `name` | Full car model name (e.g. `Maruti Swift VXI`) |
| `year` | Year of purchase |
| `selling_price` | Resale price in ₹ (target) |
| `km_driven` | Odometer reading in kilometres |
| `fuel` | Petrol / Diesel / CNG / LPG / Electric |
| `seller_type` | Individual / Dealer / Trustmark Dealer |
| `transmission` | Manual / Automatic |
| `owner` | First / Second / Third / Fourth & Above / Test Drive Car |

---

## 🧠 Model Details

| Property | Details |
|---|---|
| Algorithm | Linear Regression |
| Library | scikit-learn |
| Train/Test split | 80% / 20% |
| Feature engineering | Car age (`2026 − year`), brand extracted from name |
| Encoding | One-hot encoding via `pd.get_dummies` |
| Evaluation metrics | MAE, R² Score |

**Features used for training:**

- `km_driven` — continuous
- `car_age` — derived from year
- `fuel` — one-hot encoded
- `seller_type` — one-hot encoded
- `transmission` — one-hot encoded
- `owner` — one-hot encoded
- `brand` — first word of car name, one-hot encoded

## 🖥️ How It Works

```
User selects car model
        │
        ▼
Brand auto-extracted from name (e.g. "Maruti Swift" → "maruti")
        │
        ▼
Input row built using columns.pkl (all zeros, then flags set)
        │
        ▼
model.pkl predicts selling price
        │
        ▼
Result shown as banner + gauge + market comparison + similar listings
```

## 🔧 Customisation

**Change the base year** — the car age is calculated as `2026 − year`. Update this in both `train_model.py` and `app.py` if needed.

**Swap the model** — Replace `LinearRegression` in `train_model.py` with any scikit-learn compatible regressor (e.g. `RandomForestRegressor`) and re-run the script. The app requires no changes.

**Add more features** — Extend the dataset columns and retrain. The one-hot encoding handles new categories automatically.
